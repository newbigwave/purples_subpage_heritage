# Purples Subpage Heritage (퍼플스 서브페이지 리뉴얼)

정통 상류층 결혼정보회사 **퍼플스(Purples)**의 브랜드 가치와 품격을 현대적인 디자인으로 재해석한 서브페이지 리뉴얼 프로젝트입니다.

## 🚀 프로젝트 개요

- **목적**: 기존 서브페이지의 디자인 고도화 및 사용자 경험(UX) 개선
- **대상 페이지**:
  - 퍼플스 전통 (Heritage) - 메인 인덱스
  - 럭셔리 파티 (Party)
  - 수상 내역 (Prize)
- **주요 특징**: 
  - 고급스러운 다크 모드 및 골드 포인트 컬러 적용
  - 부드러운 스크롤 애니메이션 및 타이포그래피 강화
  - 반응형 디자인 지원 (Desktop/Mobile)

## 🛠 기술 스택

- **Core**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Animations**: CSS Transitions, Scroll-triggered animations
- **Fonts**: Pretendard (Dynamic Subset)

## 📂 폴더 구조

```text
├── index.html                  # 퍼플스 전통 (Heritage) 메인 페이지
├── AI_Antigravity_party/       # 럭셔리 파티 관련 페이지 및 자산
│   └── party.html
├── AI_Antigravity_prize/       # 수상 내역 관련 페이지 및 자산
│   └── prize.html
├── assets/                     # 공통 이미지, 로고 및 플로팅 바 스타일
├── scripts/                    # 공통 및 페이지별 자바스크립트
└── styles/                     # 공통 및 페이지별 스타일시트
```

## 💻 로컬에서 확인하기

1. 이 저장소를 클론하거나 ZIP 파일로 다운로드합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 상단 네비게이션을 통해 각 서브페이지로 이동할 수 있습니다.
